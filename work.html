Meet App

A serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events.
Key features: Filter events by city, show/hide event details, specify number of events, accessible when offline. View a chart showing the number of upcoming events by city. You are able to see a chart showing the upcoming events in each city.

Technologies used: JavaScript, React, Google Calendar API, AWS lambda, Jest / Enzyme, Jest-Cucumber, Puppeteer, Recharts.
